# Keyball44 Typing Practice Guide

A progressive typing trainer designed specifically for the Keyball44 keyboard with Miryoku layout.

## Overview

The typing practice mode helps you learn touch typing on your Keyball44 keyboard through structured lessons that progressively introduce new keys and concepts.

**Access the trainer:** [https://pegre94.github.io/zmk-config-Keyball44/](https://pegre94.github.io/zmk-config-Keyball44/)

## Features

- **Progressive Lessons** - Start with home row and gradually add more keys
- **5-Star Rating System** - Earn stars based on speed and accuracy
- **Real-time Feedback** - See your WPM, accuracy, and errors as you type
- **Keyboard Visualization** - The next key to press is highlighted on the keyboard
- **Layer-specific Lessons** - Practice NUM, SYM, and NAV layers
- **Progress Tracking** - Your progress is saved in your browser

## Tips for Success

### Posture
- Sit up straight with feet flat on the floor
- Keep wrists elevated, not resting on the desk
- Position keyboard at elbow height

### Technique
- **Don't look at the keyboard** - Trust your muscle memory
- **Use the correct fingers** - Each key has an assigned finger
- **Return to home row** - After pressing any key, return fingers to home position
- **Start slow** - Accuracy is more important than speed initially

### Practice Schedule
- **Daily practice** - 10-15 minutes per day is better than 1 hour once a week
- **Consistency** - Practice at the same time each day to build habit
- **Don't skip lessons** - Each lesson builds on the previous one

### Miryoku-Specific Tips
- **Home row mods take time** - The hold-tap behavior requires practice
- **Layer access** - Practice holding thumb keys smoothly
- **Combos** - Once comfortable, practice modifier + key combinations

## Troubleshooting

### Keys not registering?
- Make sure the browser window is focused
- Check that your Keyball44 is connected and working
- Try refreshing the page

### Layer detection not working?
- Ensure you have the latest firmware with F-key signals
- Hold the layer key before pressing other keys

### Progress not saving?
- Check that cookies/local storage is enabled
- Try a different browser
