# Keyball44 Miryoku QWERTY Configuration

Miryoku-style layout adapted for Keyball44 with trackball support.

---

## Layer Definitions

| Layer # | Name | Purpose |
| 0 | BASE | QWERTY + home row mods (GACS) |
| 1 | NAV | Navigation (arrows, home/end, page) |
| 2 | MOUSE | Mouse buttons + trackball |
| 3 | MEDIA | Media controls + Bluetooth |
| 4 | NUM | Numpad-style numbers |
| 5 | SYM | Symbols |
| 6 | FUN | Function keys |
| 7 | SCROLL | Trackball scroll mode |
| 8 | SNIPE | Trackball precision mode |

---

## Layer 0: BASE (QWERTY)

```
┌────────┬────────┬────────┬────────┬────────┬────────┐          ┌────────┬────────┬────────┬────────┬────────┬────────┐
│  ___   │   Q    │   W    │   E    │   R    │   T    │          │   Y    │   U    │   I    │   O    │   P    │  ___   │
├────────┼────────┼────────┼────────┼────────┼────────┤          ├────────┼────────┼────────┼────────┼────────┼────────┤
│  ___   │  A/GUI │ S/ALT  │ D/CTRL │ F/SHFT │   G    │          │   H    │ J/SHFT │ K/CTRL │ L/ALT  │ '/GUI  │  ___   │
├────────┼────────┼────────┼────────┼────────┼────────┤          ├────────┼────────┼────────┼────────┼────────┼────────┤
│  ___   │   Z    │   X    │   C    │   V    │   B    │          │   N    │   M    │   ,    │   .    │   /    │  ___   │
├────────┼────────┼────────┼────────┼────────┼────────┘          └────────┼────────┼────────┼────────┼────────┼────────┤
│  ___   │  ___   │ESC/MED │SPC/NAV │TAB/MOU │                   │ENT/SYM │BSPC/NUM│        │        │        │  ___   │
└────────┴────────┴────────┴────────┴────────┘                   └────────┴────────┴────────┴────────┴────────┴────────┘
```

**Home row modifiers - GACS order (tap/hold):**
- `A/GUI` = A (tap) / Left GUI (hold)
- `S/ALT` = S (tap) / Left Alt (hold)
- `D/CTRL` = D (tap) / Left Ctrl (hold)
- `F/SHFT` = F (tap) / Left Shift (hold)
- `J/SHFT` = J (tap) / Right Shift (hold)
- `K/CTRL` = K (tap) / Right Ctrl (hold)
- `L/ALT` = L (tap) / Right Alt (hold)
- `'/GUI` = ' (tap) / Right GUI (hold)

**Thumb keys - Miryoku style (tap/hold):**
- `ESC/MED` = Escape (tap) / MEDIA layer (hold)
- `SPC/NAV` = Space (tap) / NAV layer (hold)
- `TAB/MOU` = Tab (tap) / MOUSE layer (hold)
- `ENT/SYM` = Enter (tap) / SYM layer (hold)
- `BSPC/NUM` = Backspace (tap) / NUM layer (hold)
- `DEL/FUN` = Delete (tap) / FUN layer (hold) ← on trackball key

## Layer 8: SNIPE

Transparent layer - trackball switches to precision/snipe mode (400 CPI) while held.

## Legend

| Symbol | Meaning |
|--------|---------|
| `___` | Transparent (uses key from layer below) |
| `LCLICK` | Left mouse click |
| `RCLICK` | Right mouse click |
| `MCLICK` | Middle mouse click |
| `MS_L/D/U/R` | Mouse movement keys |
| `WH_L/D/U/R` | Mouse wheel keys |
| `BT_CLR` | Clear Bluetooth pairings |
| `BT 0/1/2/3` | Select Bluetooth profile |
| `UNDO/CUT/COPY/PASTE/REDO` | Standard shortcuts |
| `PREV/NEXT/PLAY/STOP` | Media controls |
| `VOL_UP/VOL_DN/MUTE` | Volume controls |
